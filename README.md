# Customer_Behaviour_analysis
Customer Shopping Data Analysis – An SQL-based data analysis project that explores customer purchase behavior, revenue trends, product ratings, discount usage, and subscription impact to generate actionable business insights.
Customer Shopping Data Analytics Project
Project Overview

This project analyzes customer shopping data to understand purchasing behavior, product performance, and revenue trends. The project involves loading and cleaning the dataset using Python, performing exploratory data analysis (EDA), running analytical SQL queries in PostgreSQL, and building an interactive dashboard in Power BI.

The final insights are presented through a professional report and presentation created using Gamma.

The goal of this project is to demonstrate a complete data analytics workflow from raw data to business insights.

Tools & Technologies

Python – Data loading, cleaning, and EDA

Pandas, NumPy, Matplotlib, Seaborn – Data analysis and visualization

PostgreSQL – Running analytical SQL queries

Power BI – Interactive dashboard creation

Gamma – Presentation creation

Jupyter Notebook – Analysis environment

Project Workflow
1. Data Loading

The dataset was loaded using Python into a Pandas DataFrame. The cleaned dataset was then stored in a PostgreSQL database for further analysis.

2. Data Cleaning

Data preprocessing steps included:

Handling missing values

Fixing inconsistent data types

Removing duplicates

Standardizing column formats

3. Exploratory Data Analysis (EDA)

EDA was performed to understand the structure and trends in the dataset, including:

Purchase amount distribution

Customer demographics

Product performance

Review ratings

Visualizations were created to identify patterns and trends.

4. SQL Analysis (PostgreSQL)

SQL queries were used to extract business insights such as:

Revenue by customer gender

Top-rated products

Discount usage analysis

Spending comparison between shipping types

Subscriber vs non-subscriber spending behavior

5. Power BI Dashboard

An interactive dashboard was created in Power BI to visualize key insights.

Dashboard highlights:

Revenue trends

Customer segmentation

Product performance

Discount impact analysis

The dashboard allows users to filter and explore the data interactively.

6. Reporting & Presentation

The insights from the analysis were summarized in a professional report and presentation created using Gamma to communicate the findings clearly.

Dashboard Insights (Key Results)

Some important insights from the analysis include:

Certain products receive consistently higher customer ratings.

Customers who use discounts often still spend above the average purchase amount.

Subscribers tend to generate higher total revenue compared to non-subscribers.

Shipping type influences purchase spending patterns.

These insights can help businesses improve marketing strategies, product recommendations, and customer engagement.
